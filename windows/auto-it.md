- **F5** : build and run script
- **Ctrl-q** (comment out selected block)
- **#comments-start -> #comments-end**
- **#cs -> #ce**
- **Ctrl-f2** (toggle bookmark)
- **Ctrl-shift-click** on region bar (will expand or close all)
